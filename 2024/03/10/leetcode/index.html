238. 除自身以外数组的乘积

利用索引左侧所有数字的乘积和右侧所有数字的乘积（即前缀与后缀）相乘得到答案，即对于给定索引 ，我们将使用它左边所有数字的乘积乘以右边所有数字的乘积。

在 O(1) 的额外空间复杂度内完成：使用res数组存储中间计算的变量。从左到右的遍历依次存储前缀，而在从右到左的遍历中每个位置乘对应的后缀。

206. 反转链表

首先定义一个cur指针，指向头结点，再定义一个pre指针，初始化为null。

然后就要开始反转了，首先要把 cur->next 节点用tmp指针保存一下，也就是保存一下这个节点。

为什么要保存一下这个节点呢，因为接下来要改变 cur->next 的指向了，将cur->next 指向pre ，此时已经反转了第一个节点了。

接下来，就是循环走如下代码逻辑了，继续移动pre和cur指针。

最后，cur 指针已经指向了null，循环结束，链表也反转完毕了。 此时我们return pre指针就可以了，pre指针就指向了新的头结点。

234. 回文链表

若使用O(1)空间复杂度解决问题，需要将链表的后半部分翻转：使用快慢指针找到链表中间节点以及尾节点，反转后将前后两部分进行比较，比较后还原链表。

参考代码如下：

class Solution {
public:
    bool isPalindrome(ListNode* head) {
        if (head == nullptr) {
            return true;
        }

        // 找到前半部分链表的尾节点并反转后半部分链表
        ListNode* firstHalfEnd = endOfFirstHalf(head);
        ListNode* secondHalfStart = reverseList(firstHalfEnd->next);

        // 判断是否回文
        ListNode* p1 = head;
        ListNode* p2 = secondHalfStart;
        bool result = true;
        while (result && p2 != nullptr) {
            if (p1->val != p2->val) {
                result = false;
            }
            p1 = p1->next;
            p2 = p2->next;
        }        

        // 还原链表并返回结果
        firstHalfEnd->next = reverseList(secondHalfStart);
        return result;
    }

    ListNode* reverseList(ListNode* head) {
        ListNode* prev = nullptr;
        ListNode* curr = head;
        while (curr != nullptr) {
            ListNode* nextTemp = curr->next;
            curr->next = prev;
            prev = curr;
            curr = nextTemp;
        }
        return prev;
    }

    ListNode* endOfFirstHalf(ListNode* head) {
        ListNode* fast = head;
        ListNode* slow = head;
        while (fast->next != nullptr && fast->next->next != nullptr) {
            fast = fast->next->next;
            slow = slow->next;
        }
        return slow;
    }
};

141. 环形链表

使用快慢指针法，若存在环则快慢指针一定会相遇。

代码：

class Solution {
public:
    bool hasCycle(ListNode *head) {
        ListNode* fast = head;
        ListNode* slow = head;
        while(fast && fast->next){
            fast = fast->next->next;
            slow = slow->next;
            if(slow == fast) return true;
        }
        return false;
    }
};

142. 环形链表 II

在快慢指针的基础上进行推导计算：

相遇时： slow指针走过的节点数为: x + y， fast指针走过的节点数：x + y + n (y + z)，n为fast指针在环内走了n圈才遇到slow指针， （y+z）为 一圈内节点的个数A。

因为fast指针是一步走两个节点，slow指针一步走一个节点， 所以 fast指针走过的节点数 = slow指针走过的节点数 * 2：

(x + y) * 2 = x + y + n (y + z)

两边消掉一个（x+y）: x + y = n (y + z)

因为要找环形的入口，那么要求的是x，因为x表示 头结点到 环形入口节点的的距离。

所以要求x ，将x单独放在左面：x = n (y + z) - y ,

再从n(y+z)中提出一个 （y+z）来，整理公式之后为如下公式：x = (n - 1) (y + z) + z 注意这里n一定是大于等于1的，因为 fast指针至少要多走一圈才能相遇slow指针。

这意味着分别从头节点和相遇节点出发一个指针，他们相遇的位置就是入口节点。

148. 排序链表

要求：在 O(n log n) 时间复杂度和常数级空间复杂度下，对链表进行排序

针对链表排序，使用归并排序的做法。用快慢指针法找到链表的中点并对链表进行递归的划分，并同样采用递归的方法对子链表排序后进行合并，最终得到完整排序的链表。

代码如下所示：

class Solution {
public:
    ListNode* sortList(ListNode* head) {
        if(!head || head->next == NULL) return head;
        ListNode* fast = head,*slow = head,*mid = head;
        while(fast && fast->next){
            mid = slow;
            slow = slow -> next;
            fast = fast -> next-> next;
        }
        mid -> next = NULL;
        return merge(sortList(head),sortList(slow));
    }
    ListNode* merge(ListNode* list1, ListNode* list2){
        if(!list1) return list2;
        if(!list2) return list1;
        if(list1->val < list2->val){
            list1 -> next = merge(list1->next,list2);
            return list1;
        }else{
            list2 -> next = merge(list1,list2->next);
            return list2;
        }
    }
};

146. LRU 缓存

使用哈希表和双向链表来实现LRU缓存，每次更新的节点都放在链表头部，最久没用的节点放在链表尾部。并添加一个空的头节点和尾节点用于标志。

代码如下：

class Node{
public:
    int value;
    int key;
    Node* prev;
    Node* next;
    Node(int key,int val): key(key),value(val), prev(nullptr), next(nullptr){}
};

class LRUCache {
private:
    unordered_map<int, Node*> cache;
    int capacity;
    int size;
    Node* dummy;
    Node* tail;
public:
    LRUCache(int capacity) : capacity(capacity) , size(0){
        dummy = new Node(0,0);
        tail = new Node(0,0);
        dummy -> next = tail;
        tail -> prev = dummy;
    }
    void remove(Node *x){
        x->prev->next = x->next;
        x->next->prev = x->prev;
    }
    void addnode(Node *x){
        x->prev = dummy;
        x->next = dummy->next;
        dummy->next = x;
        x->next->prev = x;
    }
    int get(int key) {
        auto it = cache.find(key);
        if(it == cache.end()){
            return -1;
        }
        Node* tmp = it->second;
        remove(tmp);
        addnode(tmp);
        return tmp -> value;
    }    
    void put(int key, int value) {
        auto it = cache.find(key);
        if(it != cache.end()){
            Node* tmp = it->second;
            tmp -> key = key;
            tmp -> value = value;
            remove(tmp);
            addnode(tmp);
        }else{
            Node* tmp = new Node(key,value);
            cache[key] = tmp;
            addnode(tmp);
            size++;
            if(size > capacity){
                cache.erase(tail->prev->key);
                remove(tail->prev);
                size--;
            }
        }      
    }
};
